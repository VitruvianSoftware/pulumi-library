# Vitruvian Software Pulumi Library (Public)

This repository contains the centralized "ComponentResources" for building Vitruvian Software's Google Cloud infrastructure. It is a production-grade, modular Go library ported from the Google Cloud Foundation Toolkit.

## Purpose
To provide reusable, testable, and highly-standardized building blocks for GCP infrastructure, including:
- **Project Factory**: Automated project creation and API enablement.
- **Networking**: Shared VPC and Service Networking components.
- **IAM**: Granular IAM management at all levels.
- **Policy**: Organization policy enforcement.

## Usage
Add this library to your Go project:
```bash
go get github.com/VitruvianSoftware/pulumi-library
```
